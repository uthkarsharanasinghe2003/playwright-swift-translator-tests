# Playwright UI & Functional Test Automation

## Project Description
This repository contains automated UI and functional test cases implemented using Playwright for the Swift Translator web application.

## Prerequisites
- Node.js (v16 or later)
- npm or yarn
- Visual Studio Code (recommended)

## Installation Steps
1. Clone the repository:
   git clone <YOUR_GIT_REPO_LINK>

2. Navigate to the project folder:
   cd playwright-project

3. Install dependencies:
   npm install

## Running the Tests
Run all tests:
   npx playwright test

Run tests in UI mode:
   npx playwright test --ui

## Test Structure
- tests/ : Contains all Playwright test cases
- playwright.config.js : Playwright configuration file
